## 选区工具

::: important

可选择存储和读取选区

:::

`Ctrl`+`D`取消选区

| 名称         | 备注                       |
| ------------ | -------------------------- |
| 矩形选区工具 |                            |
| 圆形选区工具 |                            |
| 多边形工具   | 多边形工具适合直线         |
| 套索工具     | 套索工具适合区块明显的图像 |
| 磁性套索工具 | 磁性套索工具很少使用       |
| 魔棒工具     | 魔棒工具可☑️ 连续，注意容差 |
| 快速选择工具 |                            |
| 选择并遮住   | CS6旧版本为调整边缘        |

::: info 布尔运算

`Shift`(`control`)添加

`Alt`(`option`)减少

:::
