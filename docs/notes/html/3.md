## 图片

```html
<img src="/2.jpg">
```

:::: field-group

::: field name="width"  optional type="数值/百分比" 

图片宽度

:::

::: field name="height" optional type="数值/百分比" 

图片高度

:::

::: field name="alt" optional type="''" 

代替图片显示的内容

:::

::: field name="title" optional type="''" 

悬停文本

:::

::: field name="align" optional type="bottom/left/right/top" 

对齐方式

:::

::::
