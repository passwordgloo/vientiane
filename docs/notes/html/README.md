## Web标准

W3C 组织负责制定 Web 全球广域网标准，标准通常包含三个方面：

*[W3C]: World Wide Web Consortium

1. 结构标准（HTML）
2. 表现标准（CSS）
3. 行为标准（JS）

*[HTML]: HyperText Markup Language

*[CSS]: Cascading Style Sheets

*[JS]: JavaScript

## 专有名词

- 标记： 比如`<p>`称为开始标记 ，`</p>`称为结束标记，也叫标签。每个标签都规定好了特殊的含义。
- 元素：比如`<p>内容</p>`称为元素.
- 属性：给每一个标签所做的辅助信息。
- HTTP：超文本传输协议。用来规定客户端浏览器和服务端交互时数据的一个格式。
- SMTP：邮件传输协议
- FTP：文件传输协议。
