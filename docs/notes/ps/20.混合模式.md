图层混合模式来说，上面图层是混合色，下面图层是基色，混合色是将混合模式效果应用到基色上

## 应用图像命令

::: note

适用于跨文档图层、通道

:::

选中图层，作为基色

图像>应用图像，源里可自定义图层或通道为混合色

选择混合模式即可

## 计算

::: note

适用于跨文档图层、通道

:::

不能选择RGB通道，计算结果是创建Alpha通道或选区

## 基础型混合模式

正常，只能调整不透明度和填充

溶解，结果色被基色或混合色随机替换，形成颗粒状过渡效果

## 变暗型混合模式

变暗，较暗的颜色为结果色，替换掉原先亮的像素，解释见P219
