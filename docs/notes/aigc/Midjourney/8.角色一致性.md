## --cref

> [!warning]
>
> 记住按空格键`--cref URL`
>
> 该命令生图只能出现相似结果，不能和参考图绝对一致
>
> 尽量使用Midjourney 自己生成的角色，还原度更好

示例

```TXT
/imagine prompt English bulldog in a pixarstyle reading a book on thebeacl --cref url --ar 16:9 --s 250
```
